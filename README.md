# SCLI (ShortCLI)

## Usage:
```console
Short Link Command Line Interface

Usage: scli --endpoint <ENDPOINT> <COMMAND>

Commands:
  get     Get the long link corresponding to the short link
  list    List short links
  create  Create a short link
  update  Update a short link
  delete  Delete a short link
  help    Print this message or the help of the given subcommand(s)

Options:
  -e, --endpoint <ENDPOINT>  Backend Endpoint
  -h, --help                 Print help
  -V, --version              Print version
```
